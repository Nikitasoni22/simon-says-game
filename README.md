# 🎮 Simon Says Game

A fun and interactive memory-based game built using JavaScript.

## 🚀 Features

* Random color sequence generation
* Increasing difficulty levels
* Sound + visual feedback
* Score tracking

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 🎯 How to Play

1. Press any key to start
2. Watch the sequence carefully
3. Repeat the sequence correctly
4. Game gets harder with each level
